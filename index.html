Maintenance status

  • ACTIVELY MAINTAINED: package supports at least the second latest Python release and is last updated less than 180 days ago
  • MODERATELY MAINTAINED: package supports at least the second latest Python release and is last updated between 180 and 365 days ago
  • POOR MAINTENANCE: package does not support at least the second latest Python release or is not updated in the last 365 days

Python Version Adoption Time

  • likely OK: package release found 180+ days after given Python release date
  • maybe OK: package release found 7-180 days after given Python release date
  • ❓ likely not OK: package release <7 days after given Python release date
  • ❌ not OK: no package release found after given Python release date
